# 빌드 실행 안내

## 현재 상태

빌드를 실행했습니다. Flutter 웹 빌드는 **몇 분 정도** 걸릴 수 있습니다.

## 빌드 완료 확인 방법

빌드가 완료되면 다음 명령어로 확인하세요:

```powershell
cd e:\flutter_projects\user_data\golf_stats_app

# 빌드 파일 확인
Test-Path build\web\index.html
Test-Path build\web\main.dart.js

# 상세 정보 확인
if (Test-Path build\web\index.html) {
    Write-Host "✅ 빌드 성공!"
    Get-Item build\web\index.html | Format-List Name, Length, LastWriteTime
    if (Test-Path build\web\main.dart.js) {
        $mainFile = Get-Item build\web\main.dart.js
        $sizeMB = [math]::Round($mainFile.Length / 1MB, 2)
        Write-Host "main.dart.js: $sizeMB MB"
    }
} else {
    Write-Host "❌ 빌드 파일 없음 - 빌드가 아직 진행 중이거나 실패함"
}
```

## 빌드가 성공한 경우

다음 명령어로 배포하세요:

```powershell
.\deploy.ps1
```

## 빌드가 실패한 경우

에러 메시지를 확인하고 다시 시도하세요:

```powershell
# Flutter 경로 추가
$env:PATH = "C:\src\flutter\flutter\bin;" + $env:PATH

# 클린 빌드
flutter clean
flutter pub get

# 다시 빌드
flutter build web --release --base-href /golf_snap_userdata/
```

## 예상 빌드 시간

- 첫 빌드: 3-5분
- 이후 빌드: 1-3분 (캐시 사용)

## 빌드 성공 시 생성되는 파일

- `build/web/index.html` - 웹 앱 진입점
- `build/web/main.dart.js` - 컴파일된 JavaScript (수 MB)
- `build/web/flutter.js` - Flutter 런타임
- `build/web/assets/` - 리소스 파일들
